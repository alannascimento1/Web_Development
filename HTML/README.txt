Objetivo: criar uma página pessoal com informações reais usando tags HTML.

Requisitos:
    - Possuir, no mínimo, 5 páginas HTML.
        *   index.html : parágrafos introdutórios e links para as demais páginas. Usar
            imagens.
        *   historicosimples.html : descrição das atividades/cargos que já desempenhou
            com datas, cargo ocupado e empresa onde atuou (pode inventar empregos).
            Deve usar tags HTML para listas.
        *   interesses.html : página que descreve seus principais interesses profissionais.
        *   contato.html : telefone para contato (fictício), endereço (fictício) e whatsapp
            (fictício), etc.
        *   redessociais.html : página que deve conter o link para suas redes sociais (pode
            ser fictício, mas o link deve levar para alguma página real). Usar imagens.
    - Deve usar:
        *   Tags de cabeçalho (h1, h2, h3, etc).
        *   Usar tag para linha horizontal para separar blocos de texto.
        *   Usar <div> para separar blocos de tags HTML.
        *   Atribuir ID as <div>s.
        *   Fazer uso de <img> e/ou <figure>.
        *   Usar listas.
        *   Usar âncoras (tag <a>) internas e externas.
        *   Usar tabelas.
        *   Usar accesskey e tabindex.
